# DS 4002 CS3: Bitcoin vs. Inflation Time Series Case Study

The present repository contains the directions, data, scripts, and reference materials needed to complete this case study evaluating whether Bitcoin acts as a reliable macroeconomic hedge against inflation.

## Directions
Access the Hook and Rubric PDFs to get started and understand your mission. The starting code for the time series analysis is titled `Template_Script.ipynb` and is found in the `SCRIPTS` folder. Everything that must be completed by you is noted by a `#COMPLETE` comment. You will need to handle data aggregation, test for stationarity, and construct a Vector Autoregression (VAR) model. To finish this case study, submit your final GitHub repository (elaborated in the Rubric) as a link that includes your final presentation slide deck.

## Materials
Four additional materials to guide and support you through the macroeconomic concepts and time series econometrics utilized in this case study are found in the `SUPPLEMENTARY MATERIALS` folder.
1. Theory: Bitcoin: An inflation hedge but not a safe haven (Choi & Shin, 2022).

2. Preparation: Introduction to Time Series Analysis (OpenStax).

3. Modeling: Introduction to VAR Models (MathWorks, 2026).

4. Selection: Probabilistic Model Selection with AIC, BIC, and MDL (Machine Learning Mastery).
